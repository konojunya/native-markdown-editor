# Native Markdown Editor

## usage

| command                 | action     |
| :---------------------- | :--------- |
| cmd \| ctrl + s         | Save File  |
| cmd \| ctrl + o         | Open File  |
| cmd \| ctrl + shift + n | New Buffer |
